# Ray Tracer

A CPU-based ray tracer implementation in C++ with OpenMP parallelization and SDL2 rendering.

## Features

- Phong reflection model with ambient, diffuse, and specular components
- Recursive ray tracing for reflections
- Shadow calculation
- Multi-threaded rendering using OpenMP
- Real-time display with SDL2

## Requirements

- C++11 or later
- SDL2
- OpenMP
- CMake
